Three lanes create visible pressure, so each turn is about lining up efficient trades before the command queue resolves.

Left Lane

Stall here when you need time to set up value engines.

Center Lane

Contest the middle to keep momentum and card draw online.

Right Lane

Push this lane when the opponent overcommits elsewhere.

Command Queue

  1. Deploy or reinforce a lane.
  2. Lock the next command into the queue.
  3. Resolve Command only after both sides finish reactions.

Pressure favors efficient trades over random burst turns.
