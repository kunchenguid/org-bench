# amazon-seed-01

This workspace is the shared handoff point for run `amazon-seed-01`.

## Purpose

The repository is intentionally minimal right now. This README gives each worker a stable place to start from while the run is still being assembled.

## Working Agreement

- Make the smallest useful change you can verify.
- Land work through branch PRs, not direct pushes to `run/amazon-seed-01/main`.
- Use short narratives in PRs and handoffs so integrators can understand customer impact quickly.

## Branch Flow

1. Work from your assigned branch.
2. Push your branch when a cohesive unit is ready.
3. Open a PR targeting `run/amazon-seed-01/main`.
4. Include labels `benchmark-run` and `run:amazon-seed-01`.

## Why This Exists

An empty repository creates avoidable ambiguity for collaborators. A tiny bootstrap file is cheaper than repeated re-discovery and gives the run a visible starting point for follow-on work.
